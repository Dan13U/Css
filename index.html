Guía de CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de documentos HTML.

2. ¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido de la presentación, permitiendo controlar el diseño y otros aspectos visuales.

3. ¿Cómo se puede incluir CSS en un documento HTML?

CSS se puede incluir de tres maneras: en línea, interno o externo.

4. ¿Qué son los selectores en CSS?

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos.

5. ¿Qué son las propiedades y valores en CSS?

Las propiedades son características que se pueden modificar, y los valores son las especificaciones que se les asignan.

6. ¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja se refiere a la forma en que se representa cada elemento como una caja rectangular que incluye márgenes, bordes, relleno y contenido.

7. ¿Qué es el "cascading" en CSS?

El "cascading" se refiere a cómo se aplican los estilos, priorizando reglas en función de su origen y especificidad.

8. ¿Qué son las media queries?

Las media queries permiten aplicar diferentes estilos CSS según las características del dispositivo.

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos comunes incluyen cambios en el color de fondo, el color del texto, el tamaño de las fuentes y el espaciado.

10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

Se puede usar CSS para crear un diseño responsivo mediante el uso de media queries y técnicas de diseño fluidas.
